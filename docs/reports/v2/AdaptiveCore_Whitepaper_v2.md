# 5. Learning Model

The Adaptive Core learns using **incremental reinforcement logic**:

1. **Observe** the event  
2. **Construct** a threat packet  
3. **Score** using short-term memory  
4. **Match** patterns with long-term signatures  
5. **Update** pattern clusters  
6. **Store** if required  
7. **Emit** immune response  

This mirrors biological immunity:

- **First exposure:** low defense  
- **Second exposure:** stronger defense  
- **Third exposure:** instant recognition and high rejection
